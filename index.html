What is a VLAN?

VLANs partition a single physical switch into multiple logical networks, each with its own broadcast domain.

802.1Q VLAN Tag (high level)

802.1Q inserts a 4-byte tag into Ethernet frames to carry the VLAN ID. Tag fields include TPID (0x8100) and TCI (PCP, DEI, VID).

TPID: 0x8100 | TCI: PCP(3) DEI(1) VID(12)

Access vs Trunk

Access — single VLAN per port. Trunk — carries multiple VLANs between switches (802.1Q tagging).

Sample Cisco Commands

  1. Create VLAN: vlan 10
  2. Assign access port: switchport mode access + switchport access vlan 10
  3. Configure trunk: switchport mode trunk + switchport trunk encapsulation dot1q
